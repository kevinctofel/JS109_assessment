## Loops and Iterating

### ```while``` Loops
Useful when you don't know how many iterations to loop.
While loops execute a code block until some condition is met. 
Example:
```js
let counter = 1;
while (counter <= 1000) {
  console.log(counter);
  counter = counter + 1;
}
```
