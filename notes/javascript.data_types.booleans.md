## Booleans in JavaScript

Two boolean literal values only: true and false.
These are like "on" and "off", or "0" and "-1".

Used for comparisons they return the boolean result of "true" or "false".
Example:
```js 
> 5 === 5
= true

> 100 < 99
= false
```
