Node.js REPL is a CLI enviornment to run JavaScript code one line at a time.

Example:
```javascript
$ node
> console.log("Hello, World!")
Hello, World!
undefined
> 3 + 4
7
> Control-D (or Control-C twice) to exit
$
```
